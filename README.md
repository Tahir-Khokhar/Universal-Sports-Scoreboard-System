# 🏆 Universal Sports Scoreboard System

A Django-based web application to manage and track live scores for multiple sports like Cricket, Football, Basketball, Tennis, and more.  
Users can create matches, update scores in real-time, and view match history with automatic winner calculation.

---

## 🚀 Features

- User authentication (Login / Logout)
- Create and manage matches
- Support for multiple sports
- Live score updates
- Automatic winner calculation
- Match history per user
- Clean and responsive UI using Bootstrap
- MySQL database integration

---

## 🛠️ Tech Stack

- Backend: Django (Python)
- Database: MySQL
- Frontend: HTML, CSS, Bootstrap
- Authentication: Django built-in auth system
